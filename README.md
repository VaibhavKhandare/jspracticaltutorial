# JavaScript to NestJS Ecommerce Backend Tutorial

## Quick Start
```bash
npm install
npm run start:dev
```

## Course Outline

### 01 - Data Types
Basic JavaScript data types with practical examples:
- Strings, Numbers, Booleans
- Arrays, Objects, Maps
- Constructor functions

### 02 - Functions & Array Methods
Essential functions and array operations:
- Regular vs Arrow functions
- map(), forEach(), filter(), sort()
- Spread operator

### 03 - Advanced JavaScript
Modern JavaScript concepts:
- Async/Await & Promises
- Destructuring
- Template literals
- Modules (CommonJS/ES6)
- Error handling

### 04 - NestJS Ecommerce Backend
Complete ecommerce backend with:
- Product management
- Order processing
- Shopping cart
- User management

## Running Examples
```bash
node 01-datatypes/strings.js
node 02-functions/arrow-functions.js
```

## Running Quizzes
```bash
node quizzes/01-datatypes-quiz.js
```

## Starting the Backend
```bash
cd ecommerce-backend
npm install
npm run start:dev
```
